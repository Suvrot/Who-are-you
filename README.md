# Who Are You

Персональный портфолио-сайт на **Next.js 16** + **Tailwind CSS 4**.

Демонстрация 6 полноценных SaaS-проектов с живыми ссылками на деплои.

---

## Стек

- **Next.js 16** / **React 19**
- **Tailwind CSS 4** / **shadcn/ui**
- **Framer Motion** — анимации
- **Vercel Analytics** — аналитика

---

## Разделы

- Hero / Обо мне / Навыки
- Проекты с Live Demo ссылками
- Опыт / Отзывы / Контакты
